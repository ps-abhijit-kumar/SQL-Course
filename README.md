# SQL Course Automation

## Prerequisites

- PostgreSQL
- Python 3.x

## Install

```bash
pip install -r automation/requirements.txt
```

## Run

```bash
cd automation
python main.py
```

## Output

The generated workbook is saved in:

automation/output/sql_exercise_outputs.xlsx

Each worksheet corresponds to one SQL exercise:

- Q1
- Q2
- ...
- Q14
